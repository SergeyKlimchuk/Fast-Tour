Окно заказа отеля
Это окно предназначено для просмотра информации об отелях и бронирования комнат.
Для перехода по страницам используйте кнопки "След страница" и "Пред страница" или пронумерованные кнопки перехода на определенные страницы.
Для поиска по отелям, заполните нужные вам поля в нижней части окна и нажмите кнопку "Поиск".
Для выхода в главное меню нажмите кнопку
Для перехода в корзину нажмите кнопку "Ваша корзина".
Для просмотра полной информации об отеле и бронирования комнаты нажмите кнопку "Подробнее…".
Для бронирования комнаты выберите дату заезда от, дату до и кнопку "Добавить в корзину".
