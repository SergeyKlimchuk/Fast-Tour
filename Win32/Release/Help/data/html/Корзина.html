Окно корзины
Это окно предназначено для подтверждения покупок и их отмены.
Для отмены заказа нажмите кнопку "Убрать".
Для просмотра информации о заказе нажмите кнопку "Информация".
Для изменения валюты покупки, выберите нужную вам волюту в выпадающем списке в левой нижней части окна.
Для полной очистки корзины нажмите кнопку
Для выхода в главное меню нажмите кнопку
Для подтверждения покупок нажмите кнопку "Перейти к оформлению!".
Первая стадия оформления покупки.
Выберите валюту оплаты и нажмите кнопку Следующий шаг, для отмены нажмите кнопку
Вторая стадия оформления покупки.
Выберите тип плательщика в выпадающем списке, заполните поля местоположения (страна и город) и нажмите кнопку "Следующий шаг".
Для отмены нажмите кнопку
Для перехода к предыдущему шагу нажмите кнопку "Предыдущий шаг".
Третья стадия оформления покупки.
Выберите тип карты из выпадающего списка, введите номер карточки, дату (месяц и год) окончания карты и нажмите кнопку "Следующий шаг".
Для отмены нажмите кнопку
Для перехода к предыдущему шагу нажмите кнопку "Предыдущий шаг".
