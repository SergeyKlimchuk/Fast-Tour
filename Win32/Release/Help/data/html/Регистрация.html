Окно регистрации
Это окно предназначено для создания новых аккаунтов. Для создания аккаунта заполните поле имени аккаунта/логина (не меньше 6 символов), поле пароля (не меньше 6 символов), повторно введите пароль для подтверждения корректности, поле почтового адреса (пример: primer@primer.com), ознакомитесь с соглашение пользователя, если вас удовлетворяет права и обязанности соглашения, то поставе галочку “Я ознакомился с соглашением принимаю его”. Если все поля заполнены правильно, нажмите кнопку “Создать аккаунт”.
Для выхода из окна регистрации, нажмите кнопку "Назад!".
Если вы заполнили поля некорректно, то вы увидите ошибку с указанными неверными полями.
