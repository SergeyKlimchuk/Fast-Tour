Окно заказа авиабилета
Это окно предназначено для просмотра информации о самолетах и покупки на них билет.
Для просмотра полной информации о самолете нажмите кнопку "Детали перелета".
Для покупки билета на самолет нажмите кнопку "Выбрать".
Для перехода по страницам используйте кнопки "След страница" и "Пред страница" или пронумерованные кнопки перехода на определенные страницы.
Для поиска по самолетам, заполните нужные вам поля в нижней части окна и нажмите кнопку "Поиск!".
